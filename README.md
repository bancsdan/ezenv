# ezenv

## Overview

This is a convenience wrapper package for getting common types from environmental variables.

## Install

Use `go get` to install this package.
```
go get github.com/bancsdan/ezenv
```

## Wrapper functions

* GetBool
* GetString
* GetUint8
* GetUint16
* GetUint32
* GetUint64
* GetInt
* GetInt8
* GetInt16
* GetInt32
* GetInt64
* GetFloat32
* GetFloat64
